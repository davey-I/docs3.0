Common-apt-commands

List packages




show all available packages from your repositories



apt list






show only the packages that are currently installed on your system.



apt list --installed






Find packages




find specific installed packages



apt list <packagename> --installed
# or use following to filter 
apt list --installed | grep <packagename>
# show packages that were manually installed.
apt-mark showmanual
# A lower-level command that can also show all installed packages. 
dpkg --get-selections






Update packages




upgrade a specific package



# upgrade only that single package, and only if it is installed.
apt install --only-upgrade <packagename>
#If you wish to install the package if it doesn't exist, or upgrade
# it if it does, you may leave out --only-upgrade.
