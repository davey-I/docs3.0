sudoers

- sudo stuff can be edited via visudo
- The sudoers policy plugin determines a user's sudo privileges.
- sudo will prompt for the password of the user specified by the -u option (defaults to root)
 instead of the password of the invoking user when running a command or editing a file.

Example Entrance



dainvirt     ALL=(ALL:ALL) ALL




     
 
  • dainvirt → The username this rule applies to.
    • 

  • ALL (before the =) → The host(s) where this rule applies. On a single server, it just means this machine.

    •  
  • (ALL:ALL) → 
       
    • First ALL → User(s) that dainvirt may run commands as (any user).

      •  
    • Second ALL → Group(s) that dainvirt may run commands as (any group).
      • 

    

    • 

  • ALL (after the )) → The commands allowed. Here it means any command.
    •  

⚡ In plain English:
dainvirt can run any command as any user and any group,
on any host, but will be asked for a password unless you add NOPASSWD: is added like so:.


dainvirt     ALL=(ALL:ALL) NOPASSWD: ALL
