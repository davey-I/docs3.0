NCDU

  ncdu (NCurses Disk Usage) is an interactive curses-based version of the well-known du(1),
  and provides a fast way to see what directories are using your disk space

Ncdu is a disk usage analyzer with a text-mode user interface. It is designed to find space hogs on a remote server where you don’t have an entire
graphical setup available, but it is a useful tool even on regular desktop systems. Ncdu aims to be fast, simple, easy to use, and should be able to run
on any POSIX-like system.

Chapter any traffic by using




  sudo tcpdump --interface any

# Limit captured packages
  sudo tcpdump -i any -c 5

sudo tcpdump -i enp0s3 -nn -s 0 -A port 443
# capture only interface enp0s3
# -i enp0s3 → listen on the correct interface
# -nn → don’t resolve hostnames or ports (keeps raw IPs and port numbers)
# -s 0 → capture full packet (default is truncated)
# -A → print ASCII contents (optional, you’ll see encrypted gibberish if using https)
# port 443 → filter for your HTTPS traffic
