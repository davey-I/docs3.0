IP-Checker-Service


Because the whole Webserversetup is based on IPV4 and we don't have a static IPV4, there is a Service that compares the Gateway IP
with the last known IP that was given to us. If it would change, this service would trigger an Alert from Statuscake.

- There is a bash script ip_checker.sh located at /home/hubi123
   that compares the current IP with the one fetched from http://ipinfo.io/ip.
   On success, it executes a CURL command to the IP-Checker Push test on Statuscake.
