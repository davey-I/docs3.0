Peter showed a way to solve the IPV6 Problem that sometimes occures on the WP-Servers
- Do : ip neighbo ( and see if there is a entrance for ::1 dev ens192 that is "STALE"
- If yes -> ip neigh del 2001:8e0:100:100::1 dev ens192
- Like that it has to show for that again, and therefore it gets renewed..
