Address-Provider-Intro

Address provider is a service developed by Tocco itself which provides a central point for all installations and answers requests in the area of addresses.

Address-Provider-Function

Basically built of three parts

  • Python: Script and request logic

  • Database as CSV file: Adresses-Data collection of provider

  • Java: For API Request stuff

  • The address-provider receives the data from an online source and then saves it in two simple CSV files city.csv & district.cs

  • The queries then get further processed by the Java-Part
