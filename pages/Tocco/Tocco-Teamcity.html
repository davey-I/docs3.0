Auto-Merge-Erstellen

- Egal ob komplett neuen Eintrag oder einen
alten wiederherstellen, folgende Docs verwenden
Wenn alten wiederherstellen, halt einfach nicht
die Ganze Anleitung durchmachen.


- Docs-Eintrag -> DOCS


- Kann alles via TC-Web-Gui gemacht werden.
Unter Projects -> Automerge -> Edit Project -> Add new Config


- Dort dann gemäss Anleitung


- Bei wiederherstellung eines alten Branches
aber nicht integration/releases/2.29 -> integration/master


Sondern einfach in die nächst höhere Branch-Version
Wenn überhaupt, da Änderungen evtl. manuell übernommen
werden müssen.


- Integration/releases/... -> integration/master
Wird logischerweise nur immer von Branch der
aktuellsten Tocco-Version erstellt
