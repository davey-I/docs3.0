Disable-Guided-Installation-Non-GUI-Installation

At the setup Window with the title 
Virtual machine Name and Operating System

We choose the folder and ISO Image + check the Skip Unattended Installation

Enable-SSH-Portforwarding

- After the machine is Shutdown, we can R-Click on the VM-Icon
  Inside the VirtualBox-Gui and go to Network Settings.

- There we can see setting for different Adapters. We choose nr.1
  And make sure its on NAT then we click to the Advanced Settings.

- Once unfolded, we can add a port forwarding like this:


Name: ssh
Protocol: TCP
HostPort: 2222
GuestPort: 22



- We should then be able to login via:


ssh -p 2222 ${VM-USERNAME}@localhost



- Additionally we can create a Matching ssh-config rule:


Host {$VMNAME}
HostName localhost
User ${VM-USERNAME}
Port 2222

Enabling-Clipboard-Sharing

- When only Server (Non-Gui) installation, we can easily go with ssh onto
the server and have we have a "shared-clipboard" for free


- Go to -> https://download.virtualbox.org/virtualbox


- Choose the right Version and download


- VBoxGuestAdditions_$VERSIONNUMBER.iso


- Oracle_VM_VirtualBox_Extension_Pack-$VERSIONNUMBER.vbox-extpack


- In Virtualbox-Gui, choose the VM,
Go to Files - Tools - Extension Pack Manager
and install the downloaded version.


- Next mount the downloaded .iso file
on to the VM.
In Virtualbox-Gui, choose the VM.
Go to Settings - Storage - Underneath Controller:IDE
you should see "empty". There we can mount the .iso


- After mounted, the we enter the VM and go to
the Tab "Devices" and choose "Insert Guest Additions CD Image"
after that, the Image should be displayed in the Tab-Section


- We can select it, a folder GUI opens, where we can choose
"RUN" via the GUI.
We probably also need to install ...

                      

    sudo apt install gcc make perl dkms
    sudo apt install bzip2
    sudo apt install linux-headers-$(uname -r)





- Reboot system and it should work..
