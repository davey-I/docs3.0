What TLS version is used by a service ? 


To find out which TLS version (and cipher suite) a web service uses,
there are several good methods â€” depending on your tools and access level.


openssl s_client -connect example.com:443



The output will show the a lot of infos also something like:


New, TLSv1.3, Cipher is TLS_AES_256_GCM_SHA384



You can also test specific versions:


# Test only TLS 1.2
openssl s_client -connect example.com:443 -tls1_2

# Test only TLS 1.3
openssl s_client -connect example.com:443 -tls1_3



If a connection fails for a given version, that version is
not supportedby the server.


We can also use these tools




nmap --script ssl-enum-ciphers -p 443 example.com
curl -v https://example.com/
